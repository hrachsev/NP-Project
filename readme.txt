================================================================================
                          SIMPLE P2P MESSENGER
================================================================================

FEATURES:
• Direct peer-to-peer messaging (no central server required)
• XML-based message format for structured communication
• Support for multiple simultaneous connections
• Simple nickname-based user identification
• Automatic friend discovery on connection

AVAILABLE COMMANDS:

MAIN MODE (when not actively chatting):
connect IP - Connect to a friend's IP address
friends - Show all connected friends
chat NICKNAME - Start chatting with a connected friend
help - Show available commands
exit - Quit the program

CHAT MODE (when actively messaging):
[type message] - Send a message to current chat partner
back - Return to main command mode

IMPORTANT NOTES:

FIREWALL: Ensure port 5299 is open on your firewall for both inbound and
outbound connections.

NETWORK: All devices must be on the same local network or have proper
routing/NAT configuration.

IP ADDRESSES: Use local IP addresses (like 192.168.x.x, 10.x.x.x, 172.16.x.x)
not public IPs for local network communication.

MULTIPLE USERS: You can connect to multiple friends simultaneously and
switch between chats using the chat NICKNAME command.

DISCONNECTIONS: The program automatically detects when friends disconnect
and removes them from your friends list.
